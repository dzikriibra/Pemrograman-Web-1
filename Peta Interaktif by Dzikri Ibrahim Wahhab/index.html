Tentang Peta Perjalanan Saya

Deskripsi Aplikasi

Aplikasi ini dibuat sebagai bagian dari tugas akhir mata kuliah Pemrograman Web. Fungsinya adalah menampilkan peta interaktif yang menunjukkan tempat-tempat yang pernah dikunjungi.

Walaupun awalnya hanya untuk memenuhi tugas kuliah, aplikasi ini dirancang dengan potensi untuk dikembangkan lebih lanjut sebagai bagian dari portofolio proyek pribadi. Dengan desain responsif dan antarmuka yang sederhana, aplikasi ini diharapkan menjadi contoh nyata dari pemanfaatan teknologi web modern dalam membangun fitur berbasis peta.

Perangkat yang Digunakan:

  • HTML – Untuk struktur dasar halaman web.
  • CSS – Untuk mengatur tampilan dan layout halaman.
  • JavaScript – Untuk logika interaktif, termasuk pengolahan data dan efek animasi.
  • Bootstrap – Framework CSS untuk responsif layout dan elemen UI seperti tombol, grid, dll.
  • Leaflet.js – Library peta interaktif untuk menampilkan lokasi-lokasi yang dikunjungi.
  • Data Statis (JavaScript) – Menyimpan daftar tempat yang dikunjungi langsung di file JS tanpa database.

Panduan Penggunaan Peta

Dokumen ini menjelaskan cara berinteraksi dengan peta interaktif, mulai dari navigasi dasar hingga fitur‑fitur lanjutan. Aplikasi ini bersifat statis untuk keperluan tugas Pemrograman Web, namun arsitekturnya dirancang agar mudah dikembangkan lebih lanjut.

Struktur Navigasi
  • About: Berisi deskripsi proyek, teknologi yang dipakai, panduan (halaman ini), dan profil pengembang.
  • Tools: Kumpulan utilitas—Locate Me, Add Location, dan Highlight Visited Places.
  • History: Membuka sidebar yang memuat daftar destinasi dalam bentuk kartu.
Navigasi Peta
  1. Gunakan scroll atau gerakan cubit untuk memperbesar/memperkecil.
  2. Klik‑dan‑seret (drag) untuk menggeser tampilan.
  3. Klik penanda (marker) untuk melihat detail singkat tempat.
Fitur Tools
  • Locate Me – Memusatkan peta ke posisi pengguna (meminta izin lokasi).
  • Add Location – Membuka formulir untuk menambahkan destinasi baru (nama, deskripsi, koordinat).
  • Highlight Visited Places – Menyorot seluruh tempat yang telah ditandai sebagai “pernah dikunjungi”.
History Sidebar

Sidebar menampilkan daftar destinasi dalam format kartu horizontal:

  • Gambar pratinjau, nama destinasi, dan ringkasan singkat.
  • Kolom pencarian di bagian atas untuk memfilter destinasi secara real‑time.
  • Klik kartu untuk menyorot lokasi yang bersangkutan di peta (dapat dikembangkan lebih lanjut).
Tips Penggunaan Efisien
  • Gunakan Locate Me sebelum menambah lokasi baru agar koordinat lebih akurat.
  • Manfaatkan fitur pencarian untuk menemukan destinasi dengan cepat saat daftar menjadi panjang.
  • Saat menambahkan deskripsi, usahakan ringkas (≤ 3 baris) agar tampilan kartu tetap rapi.

Profil Pengembang

Perkenalkan, saya Dzikri Ibrahim Wahhab, seorang mahasiswa Informatika dari Universitas Informatika dan Bisnis Indonesia (UNIBI). Perjalanan saya di dunia teknologi dimulai bukan dari rasa tahu yang luar biasa, tapi dari rasa penasaran sederhana: "Bagaimana peta di layar bisa tahu posisi kita?"

Dari situlah semuanya bermula. Perlahan saya belajar HTML, CSS, dan JavaScript. Saya tidak pernah menganggap ini mudah — justru itulah yang membuatnya menarik. Setiap bug adalah teka-teki, setiap baris kode adalah bagian dari cerita saya memahami dunia digital.

Proyek peta interaktif ini bukan hanya tugas. Ini adalah cerminan bagaimana saya belajar beradaptasi, mencoba, dan memahami teknologi dari sudut pandang pengguna dan pengembang. Karena bagi saya, prinsip hidupnya sederhana: "adaptasi atau mati."
